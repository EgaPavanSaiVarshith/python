Python provides various built-in methods and functions for working with lists. 
Modifying Lists

append(item): Adds a single item to the end of the list.

extend(iterable): Adds elements from another iterable (like a list or tuple) to the end of the current list.

insert(index, item): Inserts an item at a specified position.

remove(item): Removes the first occurrence of a specified item by value.

pop(index): Removes and returns the item at the specified index. If no index is given, it removes and returns the last item.

clear(): Removes all elements from the list.

del statement: Deletes items at a specific index or a slice.
